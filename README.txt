Приложение на языке Kotlin - PCBild. 
Приложение представляет из себя конструктор компьютерных комплектующих заданных разработчиком.
Приложение пишется для личного использования, упрощение подбора железа для моего хобби.

Задачи:
1. Создать конструктор с набором компьютерным комплектующих выбранных разработчиком. 
2. Стабильная работа приложения. 
3. Авто подбор материнской платы под размер корпуса и вид сокета процессора.
4. Составление общей картины сборки для скриншота друзьям и клиентам.

Инструменты:
1. Navigation. 
Использовал для упрощения навигации в приложение.

2. SafeArgs. 
Для передачи данных между фрагментами в Navigation.

3. Room. 
Для удобной работы с текстовой информацией о комплектующих ПК. 
В стадии реализации.

4. LiveData, ViewModel, LifeCycle. 
Для удобной работы с БД и следованию шаблону MVVM при возможности. 
В стадии реализации.

5. Возможно использование Coroutines для асинхронной работы. 
В тот случае, если будут заметны задержки в работе приложения.
